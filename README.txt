由于拓扑图在手机上实在是鸡肋，所以决定通过访问网页的形式在手机上曾现拓扑图。

在寻找开源拓扑图框架的过程中，发现了一些比较好用的库：

一、vis.js
   这是一个可视化的开源库，效果比较好，直接用cdn或者下载也行，网址：vis.js.org

二、Rapha?l
   这个主要是画矢量图形，与我要求的拓扑图不太沾边，http://dmitrybaranovskiy.github.io/raphael/，它的github地址为：https://github.com/DmitryBaranovskiy/raphael/

三、d3js也是一个不错的画拓扑图的库

四、其他的还有像jointJS、GoJS、jsPlumb等都可以，我最终使用的是vis.js
